# Personal-Portfolio-Website
🔹 Personal Portfolio Website
Created a dynamic and visually appealing portfolio to showcase my skills, projects, and contact details. Implemented modern UI techniques like split-screen design, animations, and responsive layout.

In a personal portfolio, you get to show a wide range of samples and skills and add your resume. You can add an image at the top where you can introduce yourself; below that, you can add contact information.

**Features:**
- Split-screen design
- Smooth animations and hover effects
- Sections: About, Skills, Projects, Contact
- Professional layout

## ⚙️ Steps Performed

1. Planned the structure and layout for the  project  
2. Developed the UI using HTML and CSS  
3. Implemented responsive design principles  
4. Tested functionality and fixed errors  
5. Improved UI/UX with styling and animations
